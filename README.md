**Professional landing page** for an agricultural company specializing in the purchase, storage, transportation, and **precise application of anhydrous ammonia** (a highly efficient nitrogen fertilizer).

## 🛠 Tech Stack

- **Next.js 16.2.3** — React framework (App Router)
- **React 19.2.4** + **React DOM 19.2.4**
- **TypeScript** (inferred from modern Next.js setup)

### Main Libraries & Dependencies

| Library                        | Version | Purpose                              |
| ------------------------------ | ------- | ------------------------------------ |
| **gsap**                       | ^3.14.2 | Advanced animations & scroll effects |
| **lenis**                      | ^1.3.21 | Smooth scrolling                     |
| **swiper**                     | ^12.1.3 | Image / content sliders              |
| **yet-another-react-lightbox** | ^3.30.1 | Modern image lightbox                |
| **lightgallery**               | ^2.9.0  | Alternative gallery / lightbox       |
| **next-intl**                  | ^4.9.1  | Internationalization (i18n)          |
| **sonner**                     | ^2.0.7  | Beautiful toast notifications        |
| **react-icons**                | ^5.6.0  | Icon library                         |
| **nodemailer**                 | ^8.0.5  | Contact form email sending           |

### Other Features

- Fully responsive design
- High-performance animations powered by GSAP + Lenis
- Multilingual support via `next-intl`
- Optimized image handling with Next.js
- Contact form with email functionality
- Deployed on **Vercel**
